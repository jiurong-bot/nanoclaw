2. **AI 準備完整代碼**
   - 讀取相關報告書（如需要）
   - 生成完整 JavaScript 代碼
   - 確保格式正確、縮進清晰

3. **用户自己貼上**
   - 在 Termux 中：`nano ~/nanoclaw/index.js`
   - 清空舊代碼
   - 粘貼完整新代碼
   - 保存（Ctrl+X → Y → Enter）

4. **用户啟動**
   ```bash
   cd ~/nanoclaw
   pkill -f node
   sleep 1
   npm start
   ```
