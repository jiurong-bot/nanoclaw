## 🚀 快速開始

### 1️⃣ 下載文件
```bash
# 下載 V86.0-DRIVE-complete.js
wget https://github.com/jiurong-bot/nanoclaw/raw/master/releases/V86.0-DRIVE-complete.js
```

### 2️⃣ 在手機 Termux 中部署
```bash
cd ~/nanoclaw
pkill -f node
sleep 1

# 編輯 index.js
nano index.js

# 在 nano 中：
# Ctrl+A → Delete → Ctrl+Shift+V (粘貼完整代碼)
# Ctrl+X → Y → Enter (保存)

npm start
```

### 3️⃣ 在 Telegram 中授權
```
/gauth
# 點擊授權鏈接登錄 Google 帳戶
```

### 4️⃣ 開始使用
```
/help          # 查看所有命令
/drive list    # 列出 Drive 文件
"備份我的記憶"  # 自然語言對話
```

## 🔧 配置要求

### 環境變量 (.env)
```env
TELEGRAM_BOT_TOKEN=your_bot_token
GROQ_API_KEY=your_groq_key
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
TAVILY_API_KEY=your_tavily_key
```

### Node.js 依賴
```bash
npm install
```

### 文件結構
```
~/nanoclaw/
├── index.js (本代碼)
├── .env (配置文件)
├── config/memory.json (數據庫)
├── backup_*.json (備份文件)
└── releases/ (代碼發布)

~/.openclaw/workspace/memory/
├── topics/

│   ├── 01-nanoclaw-project.md
│   ├── ... (13 個 topic 文件)
```

## 🐛 故障排除

### 問題：Google 未授權
**解決**：執行 `/gauth`，點擊鏈接授權

### 問題：Drive 文件操作失敗
**解決**：確保 `.env` 中有有效的 GOOGLE_CLIENT_ID/SECRET

### 問題：Bot 不回應
**解決**：
```bash
pm2 logs nanoclaw  # 查看日誌
npm start          # 重新啟動
```
