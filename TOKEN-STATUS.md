## 預警設定

⚠️ **當以下情況發生時，我會立即通知你：**

1. **Groq API** — 日用量超過 80% （如果有限制）
2. **Tavily API** — 月度配額超過 50%
3. **任何 API** — 出現 429（超限）或 401（認證失敗）

---

## 應急方案

如果 token 用完：

1. **切換模型** — Groq 有多個免費模型可用
   - `llama-3.3-70b-versatile` （當前）
   - `mixtral-8x7b-32768`
   - `gemma2-9b-it`

2. **臨時限制** — 降低回應長度或搜尋頻率

3. **申請新額度** — 聯絡各 API 供應商

---

## Session 恢復清單

**新 session 時，檢查這些：**

- [ ] Bot 是否仍在 Termux 運行（`ps aux | grep node`）
- [ ] Memory.json 是否完整
- [ ] API Keys 是否仍有效
- [ ] Telegram 是否能收到消息
